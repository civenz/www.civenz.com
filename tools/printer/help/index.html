以 Chrome 浏览器为例:

下载 Chrome

[取消打印预览]
--disable-print-preview
--disable-popup-blocking
--start-maximized
--allow-file-access-from-files

[静默打印]
    - [chrome.exe --kiosk-printing]
    - 或 chrome://flags > Print Preview Registration Promos > Enabled
    - "C:\Program Files (x86)\Google\Chrome\Application\chrome.exe" --kiosk-printing --app=https://www.civenz.com/tools/printer/index.html --start-maximized

[页眉, 页脚, 边柜]
    - Ctrl + P
    - 更多设置
